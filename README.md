# 火车票管理系统

SJTU CS1951 课程大作业

## 概况

### 作业安排
本作业分为两个部分。

在第一部分中，需要实现一个基于文件的 B+ 树。

在第二部分中，需要实现一个火车票管理系统。此部分要求使用 Git 开发，维持良好的项目管理习惯。此部分的中期检查等检查方式均会通过查看登记的 Git 仓库链接，因此如果想更换仓库的链接请及时联系助教。
